# вңҰ PostMind вҖ” AI-Powered Social Publishing Platform

> _Write less. Say more. Let AI think with you._

PostMind is a next-generation social platform where users craft, schedule, and share AI-assisted posts вҖ” blending the power of generative content with real social interaction.

---

## рҹҡҖ What is PostMind?

PostMind lets users generate high-quality social content using AI by simply providing a topic and tone. Once generated, content can be enhanced, image-enriched, posted instantly, saved, or scheduled for later вҖ” all within a clean social feed where others can like, comment, share, and repost.

## рҹ§ұ Tech Stack

| Layer        | Technology                          |
|--------------|--------------------------------------|
| Frontend     | React (Vite)                        |
| Backend      | Spring Boot                          |
| Build Tool   | Maven                                |
| Database     | PostgreSQL                           |
| AI Layer     | Claude API (Anthropic)              |

---

## рҹ—„пёҸ Database Overview

| Table              | Purpose                                          |
|--------------------|--------------------------------------------------|
| `users`            | User accounts, profiles, and credentials         |
| `ai_generations`   | AI-generated content per topic/tone/platform     |
| `post`             | Published posts (manual or AI-assisted)          |
| `comments`         | User comments on posts                           |
| `likes`            | Post likes                                       |
| `save_post`        | Bookmarked/saved posts                           |
| `repost`           | Reposted content                                 |
| `follows`          | Follow/unfollow relationships                    |
| `scheduled_posts`  | Posts queued for future publishing               |

## вңЁ Core Features

- рҹӨ– **AI Post Generation** вҖ” Input a topic, pick a tone, get platform-ready content
- вңҸпёҸ **Content Enhancement** вҖ” Choose simple, enhanced, or complex rewrites
- рҹ–јпёҸ **Image Support** вҖ” Attach and edit images alongside your post
- рҹ“… **Post Scheduling** вҖ” Set a future time for auto-publishing
- рҹ’ҫ **Save Drafts** вҖ” Save posts to revisit and publish later
- рҹ‘Ҙ **Social Graph** вҖ” Follow users, see their content in your feed
- рҹ’¬ **Engagement** вҖ” Like, comment, share, and repost

## вҡЎ Getting Started

```bash
# Clone the repo
git clone https://github.com/your-username/postmind.git

# Backend
cd backend
mvn spring-boot:run

# Frontend
cd frontend
npm install
npm run dev
```

> Make sure PostgreSQL is running and your `application.properties` has the correct DB credentials.
